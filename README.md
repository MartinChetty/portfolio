# Martin Chetty Portfolio

Production-ready Next.js App Router foundation for Martin Chetty's portfolio.

## Requirements

- Node.js 20.9 or later

## Commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run format:check
```

Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_SITE_URL` to the deployed canonical URL before production deployment.
